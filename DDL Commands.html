What are DDL Commands ?

DDL stands for Data Definition Language. DDL commands are used in SQL to define, manage, and modify database objects, such as tables, indexes, and constraints. These commands allow users to create, alter, and drop database objects and define their structure and characteristics

Here are the main DDL commands :-

  • Create
  • Alter
  • Drop
    •
  • Truncate
    •

Create Command

  • Create Database :-

    Used to create a new database.

    create database database_name;
  • Create Table :- Used to create a new table along with its columns and data types. 
    CREATE TABLE table_name (
    column1 datatype constraints,
    column2 datatype constraints,
    ...
    );

    • Drop Command

    • Drop Database :-

      Used to delete an entire database along with all its tables and data.

      • DROP DATABASE database_name;
    • Drop Table :-

      Used to delete an entire table along with its data and structure.

      • DROP TABLE table_name;

    Alter Command

    • ALter Table :-

      Used to modify the structure of an existing table.

      • Add a new column
        • ALTER TABLE table_name ADD column_name datatype constraints;
      • Modify a column
        • ALTER TABLE table_name MODIFY column_name new_datatype constraints;
      • Rename a column
        • ALTER TABLE table_name CHANGE old_column_name new_column_name datatype constraints;
      • Drop a column
        • ALTER TABLE table_name DROP column_name;

      Truncate Command

      • Truncate Table :-

        This command is used to quickly delete all the data from a table.

        • TRUNCATE TABLE table_name;
