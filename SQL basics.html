What are SQL and SQL Queries ?

SQL stands for Structured Query Language and it is a non procedural language. It also let you access and Manipulate databases. SQL queries are commands used to interact with a relational database and retrieve, manipulate, or manage data stored in it. SQL is a standard language for working with relational databases, and it allows users to perform various operations on the data, such as querying, inserting, updating, and deleting records.



What are basic things we can achieve using SQL ?

  • SQL can execute queries against a database.
  • SQL can retrieve data from a database.
  • SQL can insert records in a database.
  • SQL can update records in a database.
  • SQL can delete records from a database.
  • SQL can create new databases.
  • SQL can create new tables in a database.
  • SQL can create stored procedures in a database.
  • SQL can create views in a database.
  • SQL can set permissions on tables, procedures, and views.
