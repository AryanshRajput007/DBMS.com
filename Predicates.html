What are Predicates ?

In MySQL, predicates are used in SQL queries to filter and select specific rows from a table based on certain conditions.

Here are the main Predicates :-

  • IN
  • NOT IN
  • LIKE
  • BETWEEN

So let's take an reference table named "data" to check how all the Queries will work :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    IN Predicates

    IN :-The "IN" predicate is used to specify a list of values that a column must match.

    column_name IN (value1, value2, ...);

    Now let's see how the query is written using IN Predicates :-

    SELECT * FROM data
    WHERE Department IN ('IT', 'Finance', 'Manager');

    So the output of the above code will be like :-

    Id Name Department Salary
    1 John IT 60000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    NOT IN Predicates

    NOT IN :-The "NOT IN" predicate is used to specify a list of values that a column must not match.

    column_name NOT IN (value1, value2, ...);

    Now let's see how the query is written using NOT IN Predicates :-

    SELECT * FROM data
    WHERE Department NOT IN ('IT', 'Manager');

    So the output of the above code will be like :-

    Id Name Department Salary
    2 Jane HR 55000
    3 Mike Finance 58000

    LIKE Predicates

    LIKE :-The LIKE predicate is used to search for a specified pattern in a column.

    column_name LIKE pattern;

    Now let's see how the query is written using LIKE Predicates :-

    SELECT * FROM data
    WHERE Name LIKE 'Ja%';

    So the output of the above code will be like :-

    Id Name Department Salary
    2 Jane HR 55000

    BETWEEN Predicates

    BETWEEN :-The BETWEEN predicate is used to select values within a specified range.

    column_name BETWEEN value1 AND value2;

    Now let's see how the query is written using BETWEEN Predicates :-

    SELECT * FROM data
    WHERE Salary BETWEEN 57000 AND 60000;

    So the output of the above code will be like :-

    Id Name Department Salary
    1 John IT 60000
    3 Mike Finance 58000
    4 Isagi Manager 57000
