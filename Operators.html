What are Operators ?

In MySQL, operators are special symbols or keywords used to perform operations on data or values. These operations can include mathematical calculations, comparisons, and logical operations. Operators are fundamental components of SQL queries and are used to manipulate and analyze data in various ways.

In this page we will be studying following operators :-

  • Logical Operators
  • Relational Operators

What are Logical Operators?

In MySQL, logical operators are used to combine or modify conditions in SQL queries. Logical operators allow you to create more complex conditions by combining multiple conditions together.

Here are the main Logical Operators :-

  • And
  • OR
  • NOT

    AND Operator

    AND :- The AND operator combines two or more conditions and returns true only if all conditions are true.

     
    condition1 AND condition2

    Let's take an example of a table named "data" :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Now let's see how the query is written using AND Operator :-

    SELECT * FROM data
    WHERE department = 'HR' AND salary > 50000;

    So the output will be like :-

    Id Name Department Salary
    2 Jane HR 55000

    OR Operator

    OR :- The OR operator combines two or more conditions and returns true if at least one condition is true.

    condition1 OR condition2

    Let's take an example of a table named "data" :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Now let's see how the query is written using OR Operator :-

    SELECT * FROM data
    WHERE department = 'HR' OR salary > 55000;

    So the output will be like :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    NOT Operator

    NOT :- The NOT operator negates a condition, returning true if the condition is false and false if the condition is true.

    NOT condition

    Let's take an example of a table named "data" :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Now let's see how the query is written using NOT Operator :-

    SELECT * FROM data
    WHERE NOT department = 'HR';

    So the output will be like :-

    Id Name Department Salary
    1 John IT 60000
    3 Mike Finance 58000
    4 Isagi Manager 57000


    What are Relational Operator ?

    Relational operators in MySQL are used to compare values and conditions in SQL queries. These operators help you establish relationships between data by determining if a condition is true or false.

    Here are the main Relational Operators :-

    1. Equality (=)
    2. Inequality (!= / <>)
    3. Greater Than ( > )
    4. Less Than ( < )
    5. Greater Than or Equal to ( >= )
    6. Less Than or Equal to ( <= )

    So let's take an example of a table named "data" to study all the Relational operators one by one:-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Equality Operator (=)

    Equality :- The equality operator "=" checks if two values or expressions are equal.

    Value1 = Value2;

    So let's see an example for Equality Operator :-

    SELECT * FROM data
    WHERE department = 'HR';

    So the output will be like :-

    Id Name Department Salary
    2 Jane HR 55000

    Inequality Operator (!= / <>)

    Inequality :- The inequality operators "!=" and "<>" check if two values or expressions are not equal.

    value1 != value2;

    So let's see an example for Inequality Operator :-

    SELECT * FROM data
    WHERE department != 'HR';

    So the output will be like :-

    Id Name Department Salary
    1 John IT 60000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Greater Than Operator (>)

    Greater Than :- The greater than operator ">" checks if the value on the left is greater than the value on the right.

    value1 > value2

    So let's see an example for Greater Than Operator :-

    SELECT * FROM data
    WHERE Salary > 55000;

    So the output will be like :-

    Id Name Department Salary
    1 John IT 60000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Less Than Operator (>)

    Less Than :- The less than operator < checks if the value on the left is less than the value on the right.

    value1 < value2

    So let's see an example for Less Than Operator :-

    SELECT * FROM data
    WHERE Salary < 58000;

    So the output will be like :-

    Id Name Department Salary
    2 Jane HR 55000
    4 Isagi Manager 57000

    Greater Than or Equal To Operator (>=)

    Greater Than or Equal to :- The Greater than or equal to operator ">=" checks if the value on the left is Greater Than or Equal to or equal to the value on the right.

    value1 >= value2

    So let's see an example for Greater Than or Equal to Operator :-

    SELECT * FROM data
    WHERE Salary >= 57000;

    So the output will be like :-

    Id Name Department Salary
    1 John IT 60000
    3 Mike Finance 58000
    4 Isagi Manager 57000

    Less Than or Equal To Operator (>=)

    Less Than or Equal to :- The less than or equal to operator "<=" checks if the value on the left is less than or equal to the value on the right.

    value1 <= value2

    So let's see an example for Less Than or Equal to Operator :-

    SELECT * FROM data
    WHERE Salary <= 58000;

    So the output will be like :-

    Id Name Department Salary
    2 Jane HR 55000
    3 Mike Finance 58000
    4 Isagi Manager 57000
