What are DML Command ?

DML stands for Data Manipulation Language. DML commands are used in SQL (Structured Query Language) to interact with and manipulate the data stored in a database. These commands allow users to insert, retrieve, update, and delete data from database tables.

Here are the main DML commands:

  • Insert
  • Update
  • Delete

Insert Command

  • Insert Into :- The INSERT command is used to add new rows (records) into a table.

    • For example :-

  • Single Row insertion
    • INSERT INTO employees (id, name, department, salary) VALUES (1001, 'John', 'IT', 50000);

    The output of the above code will be :-

    Id Name Department Salary
    1001 John IT 50000
  • Multiple Row Insertion
    • INSERT INTO employees (Id, Name, Department, Salary) VALUES
    (1, 'John', 'IT', 50000),
    (2, 'Jane','HR', 55000),
    (3, 'Mike', 'Finance', 58000)

    The output of the above code will be :-

    Id Name Department Salary
    1 John IT 50000
    2 Jane HR 55000
    3 Mike Finance 58000

Update Command

  • Update Table :- The UPDATE statement is used to modify existing records in a table.

    • UPDATE table_name
    SET column1 = value1, column2 = value2, ...
    WHERE condition;

    For exapmle :-

    Let's take an reference table :-

    Id Name Department Salary
    1 John IT 50000
    2 Jane HR 55000
    3 Mike Finance 58000 

    UPDATE employees
    SET Salary = 60000
     WHERE Department = 'IT';

    Then the updated table will be :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000

Delete Command

  • Delete Command :- The DELETE statement is used to remove records from a table.

    • Delete from table_name where condition;

    For example :-

    Let's take a reference table :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000
    3 Mike Finance 58000 
    Delete from employees where Department = "Finance";

    Output :-

    Id Name Department Salary
    1 John IT 60000
    2 Jane HR 55000

This is all about the DML Commands and there are some additional terms as follows :-

  1. WHERE clause :- It's used to specify a condition for filtering the rows affected by the DML operation. Without it, the command would apply to all rows in the table.
  2. SELECT clause :- It specifies which columns you want to retrieve in a SELECT statement. You can use wildcard * to select all columns.
  3. VALUES :- In INSERT INTO, VALUES is used to specify the data you want to insert into the table for each column.
  4. SET :- In UPDATE, SET is used to specify the new values for the columns you want to update.
