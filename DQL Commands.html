What are DQL Commands ?

In MySQL, Data Query Language (DQL) commands are used to retrieve data from a database.

Here are the main DQL command :-

  • Select

Select Command

  • Select :- The SELECT statement is the core DQL command in MySQL. It retrieves data from one or more tables in a database.

    • Select command can be written is various ways, according to the requirements :-

    1. Basic One :- This command is used to display a single or multiple column or you can use "*" in place of column name to display the whole table.
      2.  SELECT column_name from table_name;

      Let's take an example of a table named "data" :-

      Id Name Department Salary
      1 John IT 60000
      2 Jane HR 55000
      3 Mike Finance 58000
      4 Isagi Manager 57000

    2. Using condition and other operations :- 
      3. SELECT column1, column2, ...
      FROM table_name
      WHERE condition
      ORDER BY column_name [ASC|DESC]
      LIMIT number_of_rows;

      So first let's see some basics concept that are frequently used with Select command :-

      1. WHERE condition :- Filters rows based on a specified condition, like 'department = 'HR', to retrieve specific data.
      2. ORDER BY column_name :- Sorts query results based on the specified column, either ascending or descending.
      3. LIMIT number_of_rows :- Restricts the number of rows returned by the query to the specified count, like 'LIMIT 10'.

      So the following Queries will display the following outputs:-

      1. To display the whole table.
        2.  SELECT * from data;

        So the output of the above code will be :-

        Id Name Department Salary
        1 John IT 60000
        2 Jane HR 55000
        3 Mike Finance 58000
        4 Isagi Manager 57000

      2. To display specific column from the table.
        3.  SELECT Name,Department from data;

        So the output of the above code will be :-

        Name Department
        John IT
        Jane HR
        Mike Finance
        Isagi Manager

      3. To display data where the salary is greater than or equals to 57000.
        4.  SELECT * from data WHERE salary >= 57000;

        So the output of the above code will be :-

        Id Name Department Salary
        1 John IT 60000
        3 Mike Finance 58000
        4 Isagi Manager 57000

      4. To display table in ascending order of the salary column.
        5.  SELECT * from data ORDER BY salary ASC;

        So the output of the above code will be :-

        Id Name Department Salary
        2 Jane HR 55000
        4 Isagi Manager 57000
        3 Mike Finance 58000
        1 John IT 60000

      5. To display only 3 Rows of the table.
        6.  SELECT * from data LIMIT 3;

        So the output of the above code will be :-

        Id Name Department Salary
        1 John IT 60000
        2 Jane HR 55000
        3 Mike Finance 58000

      6. To display Name, Department and Salary from the table where salary is greater than or equals to 57000 , and the table is arranged in descending order of the salary column and only 2 rows are displayed.
        7. SELECT Name,Department,Salary from data
        WHERE salary >= 57000
        ORDER BY salary DESC
        LIMIT 2;

        So the output of the above code will be :-

        Name Department Salary
        John IT 60000
        Mike Finance 58000

    We can use select command in different ways using Logical operators,Relational operators and Ralational Algebra Operations.
